Pagination Options

Pagination widget has the following configuration options:

  • containerClass: pagination
  • pageCount: 0
  • currentPage: 1
  • pageLimit: 1
  • activeClass: active
  • disabledClass: disabled
  • previousBtn: ← Previous
  • nextBtn: Next →
  • hideEmpy: true
  • connectWith: null
<div id="myPaginationContainer> </div>

To initialize the plugin writh the following code: 

<script type="text/javascript">
	$("#myPaginationContainer").pagination({ pageCount: 20 });
<script>

Optionally currentPage can be passed to start from a specific page. If currentPage is not specified widget will start from page 1: 

<script type="text/javascript">
	$("#myPaginationContainer").pagination({ pageCount: 20, currentPage: 4 });
<script>
